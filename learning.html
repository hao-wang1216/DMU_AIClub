本科生学习路径

目标：用 6-12 个月完成从基础到科研项目的能力过渡。

1. 基础阶段（0-2 个月）

  • Python 与科学计算：Python / NumPy / Pandas
  • 数学基础：线性代数 / 概率统计
  • 深度学习入门：PyTorch 官方教程
  • 阶段目标：跑通 1 个公开仓库并提交学习笔记。

2. 研究阶段（2-5 个月）

  • 论文阅读：每周 1-2 篇，使用 阅读模板。
  • 实验复现：先复现 baseline，再做消融。
  • 问题定义：提出可验证的小问题并形成 mini proposal。

3. 项目阶段（5 个月+）

  • 竞赛实践：参与本科生竞赛并完成复盘。
  • 课题产出：技术报告 / 开源代码 / 论文草稿至少一项。
  • 工程沉淀：实验模板仓库 / 项目规范

4. 推荐资源

  • 课程：机器学习 / 深度学习 / 计算机视觉 / NLP
  • 论文列表：方向 A / 方向 B
  • 代码仓库：复现友好清单
